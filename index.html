PWA Installation

iPhone: Open in Safari. Tap the Share icon. Tap "Add to Home Screen". Name it "Command".

Mac: Open in Chrome or Edge. Click the install icon in the address bar. Or go to browser menu and click "Install Nomi Command Centre".

Firefox: PWA install not supported natively. Use Chrome or Edge on desktop.

GitHub Pages Deployment

1. Create a new repository on GitHub (e.g. nomi-command-centre).
2. Upload all 5 files: index.html, styles.css, app.js, manifest.json, service-worker.js, and the icons folder.
3. Go to Settings → Pages → Deploy from branch → main → root.
4. Your app lives at https://yourusername.github.io/nomi-command-centre.
5. On iPhone, open that URL in Safari and add to Home Screen.
